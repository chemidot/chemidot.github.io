IR 리모컨

무선 신호를 통해 아두이노를 컨트롤하는 방법에는 여러가지가 있어요. 그중에서 가장 기본이면서, 생활속에서 가장 밀접하게 쓰이는 IR(적외선) 리모컨의 사용에 대해서 알아보겠습니다. 리모컨 작동을 위해서는 기본적으로 IR신호를 내보내는 리모컨본체와, 이 본체에서 내보내는 신호를 받아들이는 IR 리시버가 필요합니다.

회로도

위 그림에서 IR 리시버에 있는 핀은 모듈이 아닌 단일 부품형태의 리시버 일때 핀의 순서입니다. 그런데 보통 아두이노 학습용으로 판매하는 IR 리시버는 브레드보드에 바로 붙일 수 있도록 조그만 PCB가 달린 모듈의 형태로 판매되는 경우가 많아요. 그런 경우에는 PCB보드에 써있는 글씨를 보고 아두이노에 연결해야 합니다.

IRremote 라이브러리 추가하기

IR 리모컨에 필요한 라이브러리는 여기에서 다운 받을 수 있습니다.

스케치 → 라이브러리 포함하기 → ZIP 라이브러리 추가에서 다운 받은 파일을 선택한 뒤

스케치 → 라이브러리 포함하기 →IRremote 를 선택하면

위 헤더파일이 스케치에 추가되며, 이렇게 라이브러리를 추가해 주어야 IR 리모컨을 사용할 수 있습니다. (아두이노 IDE에 기본적으로 포함되어있는 RobotIRremote 라이브러리는 삭제합니다.)

리모컨 버튼의 HEX값 알아내기

리모컨을 사용하기 위해서는 보유하고 있는 리모컨 각 버튼의 HEX값을 알아야 합니다. 이를 위해 우선 예제하나를 업로드합니다. 사용하는 예제는

예제 → IRremote → IRrevcDumpV2 입니다. 시리얼 모니터를 열고 IR수신기를 향하여 리모컨 버튼을 누르면 각 버튼의 HEX값과 그 밖의 고유정보가 표시됩니다. 여러가지 복잡한 값이 출력되지만, 중요한 것은 맨 마지막줄에 출력되는 unsigned int data 값이에요. 이것을 잘 적어둡니다.

혹은 아래 스케치를 사용하면 간단하게 HEX값만 출력됩니다.

실제 보유하고 있는 리모컨(XA5-14)의 정보는 다음과 같아요. 일단 리모컨 타입은 NEC타입이고,

보유하고 있는 리모컨의 종류마다 HEX값이 다르므로, HEX값은 다를 수 있습니다.

각 버튼의 Code값을 알아냈으면, 이것을 실제 스케치에서 사용할 때, 각 버튼을 정의하는 값으로 사용합니다.

이런 방법으로 각 버튼의 HEX값을 정의합니다. 버튼이 보통 10~20개 정도 되요. 귀찮아도 모두 정의를 해야합니다. 배열을 사용하는 것도 좋은 방법이 되겠네요. 그리고, 별도의 리모컨이 없으면 집에서 사용하는 리모컨을 사용해도 상관없습니다.

아래 스케치는 리모컨에서 입력한 버튼을 시리얼 모니터로 출력하는 간단한 예제입니다. 시리얼 모니터는

툴 → 시리얼 모니터를 선택하거나 Ctrl-Shift-M을 눌러 모니터 창을 띄울 수 있습니다.

sketch

리모컨 버튼에 따라 서버모터가 회전하는 스케치입니다. 간단하게 3개의 버튼만 정의하였습니다.

  • Left 버튼을 누르면 5도 감소
  • Right 버튼을 누르면 5도 증가
  • OK 버튼을 누르면 0도 위치로 이동
