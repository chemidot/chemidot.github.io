렛츠아두이노

서보모터 사용하기

2017-11-20
Arduino

모터의 종류

아두이노에서 사용하는 모터의 종류에는 3가지가 있습니다.

  • DC모터
    • 전기를 인가하면 회전력이 발생하는 가장 간단한 구조의 모터
    • 보통 선이 2가닥이며, 정밀제어 불가
  • 서보모터 (아두이노에서는 주로 TowerPro SG90를 많이 사용)
    • 기어 박스를 포함하고 있는 형태의 모터. 기어 박스에 모터드라이버, 회전센서, 모터, 제어회로가 내장
    • 보통 선이 3가닥이며, 회전수 각도 등의 움직임 제어가 가능
    • SG90은 0~180˚ 제어가 가능해요. 그러므로, 자동차 바퀴를 회전시키는 용도로는 사용할 수 없음. 단, 자동차의 진행방향을 바꾸는 역할을 할 수 있음. (회전하는 바퀴는 0~360˚ 제어가 가능한 제품을 사용해야 함)
    • 스테핑모터보다 힘이 강함
    • 아두이노UNO의 USB 전류량 제한인 500mA에 따라 1~2개의 서보모터만 사용 가능
  • 스테핑모터
    • 전기 인가시 모터 내부에서 축자체가 기어스텝을 한개씩 돌리도록 제어하여 매우 정밀하게 작동
    • 보통 선이 5가닥임
    • 서보모터보다 힘이 약함

서보모터 라이브러리 추가하기

Servo 라이브러리는 아두이노 IDE에 기본으로 포함되어 있으므로, 스케치 → 라이브러리 포함하기 →Servo 를 선택하면

가 스케치에 추가되며, 이렇게 라이브러리를 추가해 주어야 서보모터를 사용할 수 있습니다.

sketch : 서보모터 180도 회전시키기

아래 스케치는 서보모터를 180도 만큼 회전시킨 후, 다시 원래의 위치로 역회전하는 간단한 예제입니다.

  • for문을 사용하는 것과, for문을 사용하지 않고 servomotor.write(180)을 하는 것에는 조금 차이가 있어요.
    • for문을 사용하면 delay 시간에 따라서 회전 속도를 조절할 수 있지만,
    • for문을 사용하지 않으면, 180도 위치로 바로 이동한다는 것입니다.
  • servomotor.write(숫자)를 사용할때, 숫자는 ‘회전각’을 의미하는 것이 아니며, 초기 위치를 0도로 기준삼아 숫자의 각도에 위치하라는 뜻입니다. 예를들어서
    • servomotor.write(90)와 servomotor.write(30)을 연속으로 실행하면 몇도를 회전하게 될까요?
    • 90도를 회전한 뒤, 추가로 30도를 회전하여 120도 위치에 있다 → 틀렸습니다.
    • 90도의 위치로 회전한 뒤, 다시 30도의 위치로 역회전하여 돌아갑니다. 즉, 30도의 위치에 있게 되는 거에요.

sketch : 시리얼 모니터를 이용해서 각도값을 입력 받아 각도 값이 변하도록 하기

시리얼 모니터는 sketch 창의 왼쪽 상단에 있는 돋보기 모양을 누르거나, 툴 → 시리얼 모니터를 선택하거나, Ctrl-Shift-M을 눌러 모니터 창을 띄울 수 있습니다. 모니터에 0~180까지의 값을 입력하면, 지정된 각도만큼 회전합니다.

sketch : 비밀번호 확인하여 서보모터 돌리기

서보모터의 날개를 금고의 개폐장치로 사용한다면, 비밀번호 금고를 만들수 있어요. 비밀번호가 맞는지 확인한 뒤 날개를 90˚ 회전시키는 방법을 사용해봅니다.
