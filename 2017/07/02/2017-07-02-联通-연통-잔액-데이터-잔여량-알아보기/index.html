联通 연통 잔액, 데이터 잔여량 알아보기

2017-07-02
china, 스마트폰

연통을 이용하기 위해 충전한 잔액과 데이터 잔여량을 알아보기 위한 방법입니다. 联通스마트폰앱을 사용해도 되지만, 저는 10010으로 문자를 보내서 알아보는 것이 더 편한 것 같더라고요.

1. 데이터 잔여량 알아보기 : 10010으로 ‘1071’ 문자 보내기

  • (이번달에) 약정된 사용량 : 1024MB
  • (내가 현재까지 사용한) 사용량 0MB (이월 데이터 우선 사용으로 사용량이 없음)
  • 이월 데이터 중 이번달 사용량 4.16MB
  • 이월 데이터 중 이번달 사용량 제외하고 남아있는 양 908.5MB

2. 충전 잔액 알아보기 : 10010으로 ‘CXYE’ 문자 보내기

  • 현재 연체된 금액 : 142.42원
  • 충전 금액 : 140원 (2.42원이 모자라서 연체되었던 모양임)
  • 월 요금 : 136원

아래 내용은 예치금과 기부금에 관한 내용이 있는데 무슨 뜻인지, 왜 이런 내용이 나오는지는 저도 잘…

  • 가용예치금 : 0원
  • 가용환급액 : 20원
  • 동결예치금 : 40원
  • 동결환급액 : 80원

연체금액이 있다고 해서 곧바로 200원을 충전을 하고 다시 CXYE를 보내봤더니, 이렇게 바뀌네요.

  • 현재 남은 금액 : 77.58원

  • 지불 가능한 남은 금액 : 197.58원 (남았던 금액 140원 + 충전금액 200원 - 연체금액 142.42원 = 197.58원)

    그런데 위에 남은 금액과는무슨 차이가 있는지 궁금하네요.

  • 월 요금 : 136원

  • 가용예치금 : 57.58원

  • 가용환급액 : 20원

  • 동결예치금 : 40원

  • 동결환급액 : 80원

​ 아래 4가지 내용은 여전히 무엇인지 잘 모르겠습니다. -_-;

3. 데이터, 문자, 음성통화 잔여량 알아보기 : 10010으로 ‘2082’ 문자 보내기

  • 현재 사용 서비스 : 106원 서비스 - 300분 통화, 1G 데이터 제공
  • 데이터 1024MB 중 0MB 사용
  • 이월된 데이터 912.65MB 중 4.15MB 사용
  • 무료통화 300분 중 0분 사용
  • 문자 300건 중 1건 사용

4. 6개월 사용 데이터 (별도 구입 건) : 10010으로 ‘5083’ 문자 보내기

  • 현재 없음 (별도 구입 건 없음)

5. 6개월 사용 데이터 구입 방법 안내 : 10010으로 ‘5082’ 문자 보내기

  • 500MB 50원 : 10010으로’GNBNB#50’ 문자 보내기
  • 1G 80원 : 10010으로’GNBNB#80’ 문자 보내기
  • 3G 200원 : 10010으로’GNBNB#200’ 문자 보내기

그런데, 혹시 데이터가 더 필요한 경우에는 위의 방법으로 구입하지는 마세요. 즈푸바오를 이용하면 1G에 35원 정도에 구입할 수 있고, 연통에서 수시로 진행하는 행사를 이용하면 1G에 20원 정도에 구입할 수도 있으니까요.
