4G 주파수 밴드, 삼성 갤럭시 Hidden Menu

2017-09-12
china, 스마트폰

LG U+에서 쓰던 삼성 갤럭시 S4에 중국 연통 유심을 넣었더니, LTE 주파수 밴드 문제때문인지(?) 발신은 되는데, 신기하게도 착신이 안되는 문제가 발생합니다. (ㅠㅜ) 착발신 모두 안되면 모르겠는데 착신만 안되는 상황이라, 이래저래 만져보다가 정말우연히 LTE 때문이라는 것을 알게 되었는데요. LTE안테나가 잘 설치되어 있는 상하이 시내에서는 전화문자수신이 안되는데, LTE설치가 안되어서(?) 3G로만 뜨는 외곽지역에 가면 발신과 수신이 모두 잘 되는 거 아니겠어요?!? 이때부터 3G전용으로 사용하는 방법을 찾아보기 시작했습니다.

4G 주파수 밴드 문제

일단 문제를 정리해보면, 한국에서 사용하던 스마트폰은 기본적으로 FDD-LTE기반이고, 중국 연통 FDD-LTE 주파수는 Band 3 (1800MHz, KT메인주파수, SKT멀티캐리어 주파수)를 사용하는데, LGU+는 Band 1 (2100MhHz)를 사용하여 주파수가 맞지 않다는 것이 원인일 것 같습니다.

아래표를 보면 알겠지만, 중국이동은 FDD-LTE 밴드가 없어서 한국핸드폰으로 3G/4G사용은 안되고, 중국전신은 가입해본적이 없어서 잘 모르긴합니다만 FDD-LTE를 밴드3로 사용하기때문에 4G사용이 가능할 것 같네요.

  • 한국 이동통신 3사 주파수 밴드

    ​

  • 중국 이동통신 3사 주파수 밴드

(※인터넷 검색을 해보면, 어떤 블로그에는 반대로 연통이 3G에서 쓰던 Band 1을 할당받아 4G에 사용하고, 전신이 Band 3를 사용하며, 연통은 Band 3의 일부, 전신은 Band 1의 일부를 보유하고 있다고 정반대로 나와있는 것도 있는데, 이 부분은 맞지 않는 듯함. 추가확인이 필요함.)

갤럭시 S4 히든모드

어쨋거나 SKT/KT기기로는 문제가 없겠지만, 현재 가지고 있는 LGU+기기로는 수신이 안되는 문제를 해결하지 못할 듯하여, 결국 LTE 모드를 포기한채 3G 전용모드로 사용하기로 하고 모드 조정을 했습니다. 모드 조정을 위해 Hidden Menu로 들어가야 하는데, 그 방법을 적어둡니다. 기록하지 않으면 곧바로 잊어버리므로…

갤럭시 S4 키패드(전화)에서 319712358 누르고 아래의 패스워드를 누르면 히든모드로 들어갑니다.

  • LGUHiddenMenu Password : 0821
  • SKTHiddenMenu Password : 996412
  • KTHiddenMenu Password : 774632

네트워크 모드 변경

히든모드로 들어가서 Network Setting → Network Mode → WCDMA ALL 선택

이렇게 WCDMA ALL로 선택하면 LTE모드가 3G모드로 바뀌면서 LG U+에서 쓰던 갤럭시S4도 발신과 수신이 모두 정상적으로 되기 시작합니다.

히든모드의 추가 세팅모드

참고로 히든메뉴에서 설정할 수 있는 다른 세팅들을 적어보면

  • HSPA Setting

​ 히든모드로 들어가서 Network Setting → HSPA Setting → HSPA 선택

  • 데이터 네트워크 방식 선택 (우선여부)

​ 키패드에서 *147359# → 3G 모드 선택 (단, 테스트해본 결과 U+ 전화기에서는 우선여부가 선택이 되지 않습니다.)

(추가) 필리핀 이동통신 2사 LTE주파수 밴드

  • Smart : 850(band5) / 1800(band3) / 2100(band1)
  • Globe : 1800(band3)

그러므로 SKT, KT향 단말기는 Smart, Globe 모두 사용 가능하지만, LGU+향 단말기는 Smart로 사용해야할 것 같습니다.
