중국의 전기요금 계산법

2012-12-24
china, 전기요금

겨울이 되다보니 전기 사용량이 매우 많아졌습니다. 전기 계량기를 보니 충전해 둔 전기를 거의 다 사용하여, 약 1달반만에 다시 1000度(1000kwh)를 충전했어요. 어? 그런데…영수증에 찍혀있는 금액이 무려 688.3원입니다. @.@ 불과 몇달전만해도 488.3원이었는데, 688.3원이 찍혀있는 것이 무언가 잘못된게 아닌가 싶더라고요. 그래서 부동산 직원에게 이유를 문의했더니, 얼마전에 전기요금이 올라 아마 그 금액이 맞을 것이라는 이야기를 해주네요. 전기요금이 올라가는 현상은 이해가 되지만, 계산을 해보니 인상폭이 무려 41%라니요 !?!

조금 이상하다 싶어 지금까지 베이징에서 전기충전한 내역을 찾아보았어요.

  • 2월 말 : 488.3원 (1000度)
  • 4월 말 : 488.3원 (1000度)
  • 7월 말 : 488.3원 (1000度)
  • 10월 말 : 516.3원 (1000度)
  • 12월 중순 : 688.3원 (1000度)

그러고보니 10월에도 약간 올랐었더라고요. 겨울에 접어들어 전기요금 인상폭이 너무 커서 인터넷 검색을 해보았는데요. 알고보니…2012년 6월까지는 누진세없이 일괄적으로 1000도당 488.3원의 전기요금이 적용되었는데, 정말 운이 없게도(!), 우리가 북경에 거주하기 시작한 올해(2012년), 7월부터 중국에서도 누진세가 적용되기 시작했다고 합니다. ㅠㅜ. 그래서 전기 누진요금이 어떤 방식으로 적용이 되는지를 찾아보았어요. 여기저기서 텍스트로 된 문서는 쉽게 찾아지기는 했는데, 끈질긴 검색끝에 중국의 모든 시/성별 전기요금 구간표가 잘 정리되어 있는 표가 있어서 그것을 첨부해보겠습니다.

중국의 행정구역별 전기요금

각 지방별로 달리 적용되는 구간표인데, 가장 위에 베이징의 전기요금 구간표가 있습니다. 언뜻 이해가 잘 안가는 부분이 있었는데, 표를 보면서 한참을 생각해보니 어떻게 전기요금이 계산되는지 대략 감을 잡을 수 있겠더라고요. 누진요금이 적용되는 와중에 그나마 위안이 되는 것은 베이징이 다른 중국도시보다 비싼편은 아니라는 점이에요.

중국의 전기요금 계산방법

일단 중국의 전기 누진세는 전체 3구간으로 나뉘어져 있습니다.

  • 1구간 : 0.4883元/kwh (0~240kwh/월)
  • 2구간 : 0.5383元/kwh (241~400kwh/월)
  • 3구간 : 0.7883元/kwh (400kwh이상/월)

2012년의 경우

전기카드로 충전하는 집의 경우에는 월별 청구가 어렵기 때문에 전체 충전도수를 가지고 전기누진세를 적용하는 것이 원칙입니다. 그런데, 2012년 7월 1일부로 전기 누진세가 적용되었기 때문에, 2012년도에는 7월~12월까지 6개월을 기준으로 전기요금을 산정하게 되고요. 이에따라, 월 240kwh까지는 1구간에 해당하기때문에 1440kwh(240kwh×6개월)까지는 1구간에 해당하고, 1441kwh에서 2400kwh(400kwh×6개월)까지는 2구간, 2401kwh이상은 3구간이 적용되게 되는 것입니다. 조금 복잡하죠?

이 원칙을 적용해서 실제 전기요금을 계산해보면,

  • 2012년의 경우 7월부터 12월까지 6개월 동안은 충전할때마다 1000度씩 충전할 경우,
    • 1차 충전시 : 1000度×0.4883 = 488.3元
    • 2차 충전시 : 440度×0.4883 + 560度×0.5383 = 516.3元
    • 3차 충전시 : 400度×0.5383 + 600度×0.7883 = 688.3元
    • 4차 충전시 : 1000度×0.7883 = 788.3元 (4차 이후부터는 계속 788.3元입니다.)

2013년부터는

2013년부터는 1월부터 누진요금이 산정되기 시작하므로, 계산이 간단해지는데요.

  • 2013년부터는 1년 단위로 누진세가 적용될 것이므로, 2880kwh(240kwh×12)까지는 1구간에 해당하고, 2881kwh에서 4800kwh(400kwh×12)까지는 2구간, 4801kwh이상은 3구간이 적용됩니다.

    • 2013년 1월부터는, 매 충전시마다 1000도씩 충전할 경우,
    • 1차 충전시 : 1000度×0.4883 = 488.3元
    • 2차 충전시 : 1000度×0.4883 = 488.3元
    • 3차 충전시 : 880度×0.4883 +120度×0.5383 = 494.3元
    • 4차 충전시 : 1000度×0.5383 = 538.3元
    • 5차 충전시 : 800度×0.5383 + 200度×0.7883 = 588.3元
    • 6차 충전시 : 1000度×0.7883 = 788.3元 (6차 이후부터는 계속 788.3元입니다.

누적충전량으로 전기요금이 계산되다보니 아무래도 겨울로 갈 수록 전기요금이 비싸지게 되는 것처럼 느껴지게 되는 것은 어쩔 수 없을 것 같습니다.

저희집의 경우 2012년의 추세를 생각해보면 1년에 약 7000度 정도 사용하게 될 것 같은데요. 이에 따른 1년간 총 전기요금은 4174.1元정도가 예상됩니다. 환율 180을 곱하면 한국돈으로 75만원(월평균 62500원)정도이지요. (단, 저희집은 여름에 에어컨 비용이 거의 들지 않는 환경입니다.) 겨울실내온도가 너무 낮아 전기누안치 1.5kwh짜리 2개를 샀는데, 앞으로는 전기요금도 만만치 않을 것 같습니다. ㅠㅜ

계산을 모두 해보고 난 뒤 알게된 한가지 신기한 사실은…항상 0.3元이 따라다닌다는 것…

한국 전기요금과의 비교

이 정도의 전기를 한국에서 사용할 경우 얼마의 전기요금이 청구되는지 궁금하여 한국전력 사이버지점에서 전기요금을 조회해보았습니다.

오늘자 환율로 계산해보면 한국돈으로 1000kwh에 약 12만원. 1000kwh를 한달반정도 사용하였으니, 한달 사용량을 667kwh로 치면, 한달 전기요금이 약 8만원정도 되는 셈.

(추가)상하이 지역의 전기요금

(2017.01.17) 2017년부터 거주하게 될 상하이 지역의 전기요금을 예상해보았어요. 구간별 요금이 북경보다 많이 높습니다. ㅠㅜ

  • 1구간 : 0.617元/kwh (0~260kwh/월)
  • 2구간 : 0.677元/kwh (261~400kwh/월)
  • 3구간 : 0.977元/kwh (400kwh이상/월)

이에 따라, 3120kwh(260kwh×12)까지는 1구간에 해당하고, 3121kwh에서 4800kwh(400kwh×12)까지는 2구간, 4801kwh이상은 3구간이 적용되겠네요. 그러므로, 2017년 1월부터 매 충전시마다 1000度씩 충전한다고 가정하면,

  • 1차 충전시 : 1000度×0.617 = 617元
  • 2차 충전시 : 1000度×0.617 = 617元
  • 3차 충전시 : 1000度×0.617 = 617元
  • 4차 충전시 : 120度×0.617 +880度×0.677 = 669.8元
  • 5차 충전시 : 800度×0.677 + 200度×0.977 = 737元
  • 6차 충전시 : 1000度×0.977 = 977元 (6차 이후부터는 계속 977元입니다.)

이 조건으로 1년에 7000度(한달에 583kwh를 사용하는 정도)를 충전한다고 가정하면,

  • 베이징 : 488.3+488.3+494.3+538.3+588.3+788.3+788.3 = 4174.1元/1년

​ (환율 180원 적용시 한국돈으로 1년 전기사용료는 751,338원 = 62,611.5원/월)

  • 상하이 : 617+617+617+669.8+737+977+977 = 5211.8元/1년

​ (환율 180원 적용시 한국돈으로 1년 전기사용료는 938,124원 = 78,177원/월)

에효…상하이 무지 더울텐데, 베이징보다 연간 1000원 가까이 비싸네요. 그래도 한국보다는 저렴하다는 것으로 위안을 삼아야겠습니다. 상하이의 전기요금 적용은 여기를 참고하세요.
